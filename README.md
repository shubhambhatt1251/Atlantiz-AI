## 🚀 **Quick Start**

```bash
# Clone the repository
git clone https://github.com/shubhambhatt1251/Atlantiz-AI.git

# Navigate to project directory
cd Atlantiz-AI

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
