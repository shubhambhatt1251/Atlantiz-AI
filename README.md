## 🚀 **Quick Start**

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/atlantiz-ai.git

# Navigate to project directory
cd atlantiz-ai

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
